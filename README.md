Conducted exploratory data analysis on a Loan Default dataset comprising 148,670 rows and 34 features. Identified and segregated missing values by missing data type. Investigated the target column distribution and suggested appropriate handling strategies. Analyzed feature distributions through skewness and kurtosis metrics, and documented preprocessing and pipeline transformation decisions. Plotted box plots to detect outliers. Applied statistical tests — Chi-square, Information Value, Cramér's V, Variance Inflation Factor (VIF), and Mann-Whitney U — to assess feature-to-feature relationships and feature-to-target associations.
